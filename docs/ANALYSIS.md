# ANALYSE : MANGO INFLUENCE

## CARTE D'IDENTITÉ DE L'AGENCE

| Critère | Détail |
| :--- | :--- |
| **Nom** | Mango Influence (Agence de création de contenu) |
| **Ancienneté** | 3 ans d'existence |
| **Expansion géographique** | 4 implantations : DOM-TOM, Paris, Barcelone, Madrid |
| **Effectif commercial** | 6 personnes (4 commerciaux salariés + 2 agents en Espagne) |
| **Maturité organisationnelle** | RH & Gestion de projet : ✅ Structuré (audit réalisé) / Commercial & Marketing : ❌ Non structuré |
| **Budget disponible** | 15 000€ sur 4 mois (hors investissement CRM) |

## PROPOSITION DE VALEUR (Value Proposition)

"De l'idée au post : Nous créons, amplifions et mesurons l'impact de vos contenus sur tous les écosystèmes digitaux (Social Media, Influence, Branded Content)."

**Offre probable :**
1. Création de Contenu : Production vidéo, photo, motion design pour marques.
2. Marketing d'Influence : Identification et activation d'influenceurs/créateurs.
3. Gestion Social Media : Community management, stratégie éditoriale.
4. Campagnes Paid Media : Meta Ads, TikTok Ads, YouTube Ads.

## ANALYSE SWOT

### ✅ FORCES (Strengths)
- **Croissance Explosive** (3 ans = 4 bureaux internationaux)
- **Positionnement Géographique Unique** (DOM-TOM + Paris + Espagne)
- **Audit RH/Projets déjà réalisé**

### ❌ FAIBLESSES (Weaknesses)
- **Anarchie Commerciale** (6 commerciaux = 6 méthodes)
- **Absence de Playbook Commercial**
- **Pas de Vision Marketing Structurée**
- **Risque de Dilution de l'Image de Marque**

## DIAGNOSTIC : LES 3 URGENCES

1. **Pas de processus de vente unifié** -> Perte de 30-50% du CA potentiel.
2. **Pas de stratégie d'acquisition client** -> Croissance imprévisible.
3. **Pas d'outil de pilotage** -> Risque de trésorerie.

- **Mois 1**: Audit & Cartographie

- **Mois 4**: Pilotage & Optimisation
